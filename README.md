### Implements Detail
- SW
  - Object Detection using OpenCV (image processing Library)
  - Image Classification using Deep Learning (Keras)
  - user interface using Gui program (Pyqt5)

- HW
  - DC Motor for driving Conveyor belt
  - Step Motor for sorting Lego
